# Chatbot (console version)

Simple but very persuasive selling chatbot

### In order to train the model:
- cd ../the_chatbot
- python main.py

### To run the chatbot:
- cd ../the_chatbot
- python chat_bot.py

PS: try to ask it for a joke ;)
